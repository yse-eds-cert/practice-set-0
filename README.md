# Assignment 0: Introduction to Google Colab & GitHub

Welcome to the YSE Environmental Data Science Certificate Program! This assignment will walk you through the tools we'll use throughout the program: Google Colaboratory (Colab) and GitHub.

## How to Open the Assignment Notebook

1. You have already **accepted the assignment from Canvas** via the invitation link. When you did that, GitHub Classroom created your personal repo (e.g., `yse-eds-cert/assignment-0-YOURNAME`). 
2. After you accepted the assignment, you then **went to your personal repo** on GitHub (e.g., `https://github.com/yse-eds-cert/assignment-0-YOURNAME`). If you look at your browser's link, you'll see that your are now located in your personal repo! 
3. Next, you need to **Click on `assignment_0.ipynb`** — GitHub will show an **"Open in Colab"** button at the top of the file. Open that link in a new tab so that you can togle between these instructions and your opened Assignment 0. 
4. In Colab, go to **File → Save a copy in Drive** — this creates an autosaving working copy in your Google Drive. Do all your work there, as you'll always then have a copy backed up in your personal Google Drive.
5. Follow the instructions of Assignment 0 to **Complete your Assignment** using the R kernel.
8. When finished, submit by going **File → Save a copy in GitHub** → select your personal assignment repo (`yse-eds-cert/assignment-0-YOURNAME`) and push your final work to your GitHub repo.

> **Why save to Drive first?** Colab autosaves to Google Drive but does **not** autosave to GitHub. Saving to Drive protects your work-in-progress. Saving to GitHub is the manual submission step.
